Sistema hexadecimal

Sistema hexadecimal

Mientras que la dotación digital tiene diez dígitos, y es muy utilizada en nuestro día a día, la informática y el procesamiento de datos se basan en el código binario, sin embargo, cuando nos queremos referir a cantidades grandes, estos son poco claros. El sistema hexadecimal es el que se encarga de esto, la información que requiere ocho dígitos en el sistema binario, puede expresarse con solo dos números hexadecimales.

Se utiliza en la informática para facilitar la legibilidad de números grandes o secuencias de bits largas. Estas también se utilizan, entre otras cosas, en la dirección de origen y destino de protocolos de internet (IP) en los códigos ASCII o en la descripción de los códigos de color en diseño web con el lenguaje de hojas de estilos en CSS

Este sistema utiliza 16 dígitos diferentes. se utilizan los números decimales 10, 11, 12, 13, 14 y 15, los cuales se reemplazan con las letras mayúsculas A, B, C, D, E y F, y los números del 0 al 9 para representar el equivalente numérico binario o decimal. Para poder distinguir los números hexadecimales de los decimales, existen varias notaciones (por ejemplo, el número “73” en hexadecimal puede ser 73hex, entre otras formas de representarlo)
